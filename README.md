# AI Infrastructure Checklist: GitHub Pages

Мини-сайт для GitHub Pages с чек-листом по выбору AI-инфраструктуры: VPS, GPU, API, n8n, Ollama, Open WebUI, Telegram-боты и AI-агенты.

Страница сделана на чистом HTML, CSS и JavaScript. Без React, Vue, Bootstrap и внешних библиотек.

## Структура

```text
docs/
├── index.html
├── style.css
└── script.js
```

## Что внутри

- быстрый выбор между VPS, GPU, API и гибридной схемой;
- таблица минимальных конфигураций;
- короткое сравнение API и своего сервера;
- список типовых ошибок;
- базовый стек для n8n и AI-агента;
- ссылки на дополнительные материалы wturm.ru.

## Как опубликовать через GitHub Pages

1. Скопируйте папку `docs` в корень репозитория.
2. Сделайте commit и push в GitHub.
3. Откройте настройки репозитория: `Settings → Pages`.
4. В блоке `Build and deployment` выберите:
   - Source: `Deploy from a branch`;
   - Branch: `main` или ваша основная ветка;
   - Folder: `/docs`.
5. Нажмите `Save`.
6. Через некоторое время GitHub покажет ссылку на опубликованную страницу.

## Локальная проверка

Можно открыть файл напрямую в браузере:

```bash
open docs/index.html
```

Или поднять простой локальный сервер:

```bash
python3 -m http.server 8000 -d docs
```

После этого откройте:

```text
http://localhost:8000
```
