# Brief: AI Infrastructure Checklist GitHub Pages

## Задача
Подготовить мини-сайт для GitHub Pages на русском языке.

## Файлы
- `docs/index.html`
- `docs/style.css`
- `docs/script.js`
- `README.md` с инструкцией публикации из `/docs`

## Интент
Сделать полезную статичную страницу-чек-лист по выбору инфраструктуры для AI-задач: VPS, GPU, API, n8n, Ollama, Open WebUI, Telegram-боты и AI-агенты.

## Аудитория
Разработчики, фрилансеры, вебмастера и владельцы небольших проектов, которые выбирают сервер/API/GPU под AI-задачи.

## Стиль
Тёмно-синий tech-дизайн, белые карточки, таблицы, аккуратные кнопки, минимализм. Без рекламного тона и обещаний.

## Ограничения
- Чистый HTML/CSS/JS.
- Без React, Vue, Bootstrap и внешних библиотек.
- Адаптивность под мобильные.
- Все внешние ссылки открываются в новой вкладке.
- Ссылки на wturm.ru обычные, без nofollow.
- Страница должна быть полезной сама по себе.

## Допущения
- Репозиторий публикуется вручную.
- GitHub Pages будет включён из папки `/docs`.
- Обложка/social preview полезна, поэтому создан `image_prompt.txt`.
